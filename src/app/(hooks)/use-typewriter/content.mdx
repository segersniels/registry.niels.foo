# useTypewriter

A hook that simulates typing of a string.

## Usage

```tsx
import useTypewriter from "@/hooks/use-typewriter";

export default function Example() {
  const value = useTypewriter("Hello, world!", 2000);

  return <div>{value}</div>;
}
```
