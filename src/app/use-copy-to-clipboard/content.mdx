# useCopyToClipboard

A hook that simplifies copying to clipboard.

## Usage

```tsx
import useCopyToClipboard from "@/hooks/use-copy-to-clipboard";

export default function Example() {
  const { isCopied, copyToClipboard } = useCopyToClipboard();

  return <Button onClick={() => copyToClipboard("Hello, world!")}>Copy</Button>;
}
```

## Installation

```bash
npx shadcn add "https://registry.niels.foo/use-copy-to-clipboard.json"
```
